# UEA Tareas
_Soy Ximena Castro Ceja, estudio la universidad en la UAM Lerma y estoy cursando el 5to trimestre de la carrera de Artes y Comunicaciones Digitales la UEA de Comunicación creativa en la cultura digital._
## Contenido
_En este repositorio se van a presentar tareas y trabajos con códigos realizados en Visual Studio Code sobre cómo construir una página web desde cero. En esta aplicación se verán formas de cómo insertar texto, como títulos, párrafos y botones, e hipervínculos que puede contener la página, así como las formas de estilo de la página, como los colores, tipos de fuente, márgenes y diseño, además de contenido multimedia como imágenes, video y audio para la programación de tu propia página web. Además, se trabajará en un portafolio digital en línea, en la cual se podrá explorar y recopilar algunos de mis trabajos como artista digital en la carrera._
## Relación con las tecnologías web
_La relación que eh tenido con las tecnologías web desde que entre a la universidad ah sido bastante amplia, y eh descubierto distintos métodos de procesar la información que me ofrecece, támbien nuevas formas de aprender a utilizarlas, así cómo nuevas herramientas que antes desconocía, tanto cómo en la programación, el diseño, cómo utilizar la información de manera correcta y confiable para trabajos de investigación, y maneras de crear contenidos multimedia._
