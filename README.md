# UEA Tareas
_Soy Ximena Castro Ceja, estudio la universidad en la UAM Lerma y estoy cursando el 5to trimestre de la carrera de Artes y Comunicaciones Digitales la UEA de Comunicación creativa en la cultura digital._
## Contenido
_En este repositorio se van a presentar tareas y trabajos con codigos realizados en Visual Studio Code, sobre como construir una página web desde cero, en esta aplicación, se veran formas de como insertar texto, como titulos, parrafos, y botones, e hipervinculos que puede contener la página, así como las formas de estilo de la página, como los colores, tipos de fuente, margenes y diseño, ademas de contenido multimedia como imagenes, video y audio para la programación de tu propia página web._
## Relación con las tecnologías web
_La relación que eh tenido con las tecnologías web desde que entre a la universidad ah sido bastante amplia, y eh descubierto distintos metodos de procesar la información que me ofrecece, tambien nuevas formas de aprender a utilizarlas, así como nuevas herramientas que antes desconocía, tanto como en la programación, el diseño, como utilizar la información de manera correcta y confiable para trabajos de investigación, y maneras de crear contenidos multimedia._
